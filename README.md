# kotlin-lab8
##  Description
Projet Kotlin pour le TP/Lab 8 — Collections et Nullability.
Ce lab montre comment manipuler des listes, ensembles et maps en Kotlin, et comment utiliser les opérateurs in, !in, et Elvis ?:.
##  Contenu
Étape 1 : Créer la classe Comment (userId + message).

Étape 2 : Créer la liste comments: List<Comment>.

Étape 3 : Créer le set blockedUserIds: Set<Int> pour gérer les utilisateurs bloqués.

Étape 4 : Créer la map userIdToRelation: Map<Int, String> pour stocker les relations.

Étape 5 : Parcourir la liste avec for et filtrer les commentaires non bloqués.

Étape 6 : Récupérer la relation via map[key] et utiliser ?: "unknown" si absente.

Étape 7 : Afficher le commentaire final avec println("Comment ${comment.message} from $relation").

Étape 8 : Exécuter le programme et observer la sortie.
##  Utilisation
Copier le code complet dans un fichier main.kt.txt
Compiler et exécuter avec Kotlin
